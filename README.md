# reactTodoApp
A basic To Do app made with React js a javascript library.
This app has CRUD implemented and reuseable components ,so we can add tasks ,edit them ,updat and delete them.

To start with the app just download or clone the repo to your pc or laptop.
OPen the terminal and run npm install in the terminal to intstall all the necessary dependencies for the project.
Then run npm start to run the project in your browser n your localhost.
